# calculator-web-app
Calculator web app written in rust, askama, htmx, and tailwind.

## About
A simple calculator meant to further improve my understanding of rust and relevant libraries.

## Quick Start
```
git clone https://github.com/itlogsandwich/calc-stands-for-calculator
cd calc-stands-for-calculator

build the css, npx tailwindcss -i ./assets/input.css -o ./assets/output.css

cargo run -r
```
